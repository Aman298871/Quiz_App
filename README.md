# Quiz App 📚🎯

A simple and interactive Android Quiz Application developed using **Kotlin** and **XML** in Android Studio. The app allows users to answer multiple-choice questions, check their knowledge, and restart the quiz at any time.

## Features ✨

* Multiple Choice Questions (MCQs)
* Four options for each question
* Instant answer checking
* Score tracking
* Restart Quiz functionality
* Attractive quiz-themed UI
* Beginner-friendly project structure

## How It Works ⚙️

1. User launches the application.
2. A question is displayed on the screen.
3. User selects one of the four options.
4. The app checks whether the answer is correct.
5. Score is updated accordingly.
6. After all questions are completed, the final score is shown.
7. User can restart the quiz and play again.
